# DM REALM ALPHA Logger

Sistema avanzato militare di osservazione eventi per Discord.  
Tracciamento completo operazioni server.  
"**Nihil Difficile Volenti**"

---

### ✅ Funzioni
- Log join/leave
- Log messaggi cancellati e modificati
- Log ban/kick/timeout
- Log attività vocali
- Log assegnazione/rimozione ruoli
- Log inviti generati
- Embed estetici militari
- Tracking executor + trace ID

---

### ⚙️ Setup
1. Imposta `.env`
2. `npm install`
3. `npm start`
